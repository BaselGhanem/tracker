# دفتر حسابات شخصي

تطبيق ويب لتتبع الإيرادات والمصروفات والتحويلات بين الحسابات بالدينار الأردني.

## المميزات
- تسجيل الإيرادات والمصروفات
- تحويل الأموال بين الحسابات
- عرض أرصدة جميع الحسابات
- رسوم بيانية لتوزيع الإيرادات والمصروفات
- واجهة مستخدم باللغة العربية

## التقنيات المستخدمة
- React.js
- Firebase (Authentication & Firestore)
- Recharts للرسوم البيانية
- Tailwind CSS للتنسيق
